## 📌 Overview

**SignBridge AI** is a revolutionary web application designed to break down communication barriers for deaf and mute individuals. By combining real-time sign language recognition, AI-powered translation, and interactive learning — SignBridge empowers users to communicate, learn, and thrive in an inclusive digital world.

---

## ✨ Features

### 🎓 Learning Mode

- Interactive video lessons for sign language
- AI-powered practice sessions with instant feedback
- Gamified learning experience to keep users engaged

### 🔄 Translation Mode

- **Real-time sign-to-text** translation using webcam-based gesture recognition
- **Text-to-sign** demonstration for hearing users
- Dual language support for two-way communication

### 🌍 Accessibility First

- No special hardware required — just a webcam and browser
- Fully responsive design for desktop and mobile
- Privacy-focused: all video processing happens on-device

## 🚀 Getting Started

### Prerequisites

- Node.js `v18+`
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com//signbridge.git

# Navigate into the project
cd signbridge-ai

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Vercel Link

Open https://signbridge-ai-final.vercel.app

### Building for Production

```bash
npm run build
npm start
```

## 🌐 Deployment

This project is deployed on **Vercel**. To deploy your own instance:

1. Fork this repository
2. Connect your GitHub repo to [Vercel](https://vercel.com/)
3. Add any required environment variables in the Vercel dashboard
4. Click **Deploy**
